A — Grid oben, Tasche unten

Der Wochenplan nimmt den Hauptteil des Bildschirms ein. Die Auftragstasche ist eine scrollbare Karten-Leiste unten. Aufträge können in freie Slots gezogen werden. Klassisches Kanban-Prinzip.

C — Vollbild Grid + KI-Accept/Reject

Der Plan nimmt fast den ganzen Screen ein. Jeder KI-Vorschlag hat direkt ✓/✗ Buttons. Auftragstasche ist ein fester Streifen unten. Freie Slots zeigen "+ Auftrag" zum Reinziehen.
